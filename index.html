Javascript is a client side single-threaded, asynchronous, non-blocking, interpreted script language who run browser and nodeJS. Each browser have javascript engine for run javascript. Like chrome and nodeJS have V8 Engine, FireFox have spiderMonkey, internet explorer have chakra. Javascript Engine have parser who check javascript code line by line when get a error stop Execution code and through a error message. If code is perfect parser sent a deta structure AST (Abstract Syntex Tree) convert to machine code. When code is converted in machine code then javascript code run in browser.

Each browser have runtime or js engine for executing Javascript code and runtime or js engine have two parts heap and call stack. basically heap dose memory management job and call stack is like arranging books on books, when a function invoke its go to the call stack and executing function invoke another function the engine add the function to the stack. Once finished executing second function engine takes out from the stack then executing first function and remove the function from stack and continue the same process. But when engine find out asynchronous element in the stack like setTimeOut, setInerval, fetch, promise etc that time engine take out this element from the stack and give it web api or browser api and web api complete the task asynchronous callback function then puts it the callback queue. In the stack as usual execution function and pops out stack also the main() pops out of the stack in the mean time event loop recognizes the call stack is empty and callback queue have a callback function then event loop put the callback function into the call stack and nothing else is in the stack event loop execute callback queue further.

SessionStorage :- Session Storage is Unique per tab. Its session based work per window or tab. If you close the tab SessionsStorage gets deleted. It’s expired when tab is close.
LocalStorage :- It is not Session Based, it must be deleted via javascript or manually. It’s expired date is forever.

SessionStorage :- The storage capacity of session storage is 5mb.
LocalStorage :- The storage capacity of Local Storage 5 to 10 mb (browser depend).

SessionStorage :- Accessibility of session storage only same tab.
LocalStorage :- Accessibility of Local Storage Any window
